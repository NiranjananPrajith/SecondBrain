## Overview

A **tech company** (not a farming company) that develops and sells both hardware and software for indoor farming needs.

**Core Thesis:** Outdoor farming will become impossible due to climate change. Indoor controlled-environment agriculture (CEA) is the future. But indoor farming requires sophisticated tech most growers don't have access to. We fill that gap.

**Business Flip:** Our profits depend on MORE growers entering the market — not competing with them.

## Origin Story

### The Climate Reality
- Dr. Rowen grew up in a region experiencing climate change firsthand
- Unpredictable weather, tropical storms, floods, heat waves
- New pests migrating, crops failing, food shortages
- **Conclusion:** Outdoor farming will eventually become impossible

### Innovation Event (Post-School)
- Presented container farm idea at university innovation event
- Decommissioned shipping containers → grow crops inside
- Estimated yields (tomato, cabbage, cauliflower), ROI projections
- Idea appreciated but no follow-through — too risky, too much capital needed

### Pivot #1: Identified the Problem
- Low return per square cube with traditional crops
- Needed something smaller, more expensive, higher value

### Pivot #2: Indoor Saffron
- Friend proposed indoor Kashmiri saffron cultivation
- Iranian growers making huge profits, Kashmiri saffron still outdoor
- High value = high return per square cube ✓
- **Problem:** Long investment horizon + market volatility risk
- If everyone starts indoor saffron farming → supply increases → prices crash

### Pivot #3: The Flip
- **Old model:** Compete on saffron production
- **New model:** Make money ONLY if more growers enter the market
- Build the infrastructure every indoor grower needs
- → **Tech company, not farming company**

## The Business Model

### Phase 1: R&D Farms (Validation + Proof)
**Purpose:** Prove the tech works in real conditions

- Run actual indoor farms using exclusively our own technology
- Produce real crops, sell in real markets
- Document everything, generate online awareness/content
- If R&D farms turn profitable → the business model is validated
- Online buzz attracts interested growers who visit farms
- Visitors see profitable operation → want to replicate
- **Key insight:** We don't sell a dream. We sell a proven system.

### Phase 2: Sell the Tech Stack
**Products:**

#### Hardware
- **Sensors:** Temperature, humidity, plant growth tracking
- **Disease/illness scanning:** Computer vision for early detection
- **Hydroponic/aeroponic racks:** Modular, controllable
- **Automated nutrient delivery:** Precise control over feeding
- **IoT hub:** Central monitoring and control device
- **Drones (future):** Small drones for large greenhouse/outdoor monitoring
- **Embedded devices (future):** Scattered across farms for soil moisture, pH, etc.

#### Software
- **Monitoring app:** Real-time dashboard for all farm vitals
- **ML analytics:** Information aggregation, processing, data analysis
- **AI action plans:** Yield improvement, disease management, plant care recommendations
- **Control system:** Easy monitoring, controlling, analyzing entire setup

### Phase 3: Sales Options

| Option | Description |
|--------|-------------|
| **DIY** | Buy individual products, set up yourself, control with our software |
| **Turnkey** | We design + set up everything, provide training on usage and best practices |

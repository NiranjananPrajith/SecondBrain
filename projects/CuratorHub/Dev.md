# CuratorHub Dev

*The AI-Powered Development Platform*

---

## Overview

A platform where creators build applications using AI — "vibecoding" with intelligent assistance.

**Domain:** curatorhub.dev (future)

---

## The Four Spaces

```
Dashboard → Blueprint → Workshop → Assembly
    ↓          ↓          ↓          ↓
 Overview    Plan      Generate    Finish
```

### Dashboard
- All projects
- Recent applications
- Quick start templates
- Project management

### Blueprint
- Planning space
- Feature specifications
- UI/UX planning
- Architecture outlines

### Workshop
- Code generation
- AI pair programming
- Frontend + Backend generation
- Database design

### Assembly
- Combine modules
- Test & debug
- Deploy applications
- Finalize builds
